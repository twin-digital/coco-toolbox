1. Create a new file in the root directory of the project named "README.md".
2. Open the file in your preferred text editor.
3. Begin the file with a title denoted by a "# " at the start of a new line. For example: "# Coco-Toolbox".
4. Write a brief overview of the project, including its purpose and functionality.
5. Specify the technologies and dependencies used in the project.
6. Provide instructions on how to install and set up the project.
7. Explain the project structure and the purpose of each component and module.
8. Include detailed explanations for any important API endpoints or functions.
9. Provide examples of how to use the project in real-world situations.
10. Add a contributor section with instructions on how to contribute to the project.
11. Include a license section specifying the licensing terms for the project.
12. Finally, add any additional documentation or resources that could be helpful for users or contributors to the project.

# Coco-Toolbox

The Coco-Toolbox project is a collection of utilities used throughout the Twin Digital organization. This project is implemented in Typescript and is designed for high standards of maintainability, scalability, and security.

## Technologies

The following technologies are used in this project:
- Typescript
- Node.js
- NPM

## Installation

To install and set up the project, follow these steps:
1. Clone the repository from Github: `git clone https://github.com/yourusername/coco-toolbox.git`
2. Install the project's dependencies: `npm install`
3. Build the project: `npm run build`

## Project Structure

The Coco-Toolbox project is structured as follows:
- `src/` contains the source code for the project.
- `dist/` contains the compiled TypeScript code.
- `tests/` contains the unit tests for the project.

## API Endpoints

The following API endpoints are available in this project:
- `/utils/encode` - accepts a string and returns the Base64 encoded version of the string.
- `/utils/decode` - accepts a Base64 encoded string and returns the decoded string.

## Usage

The Coco-Toolbox project can be used in a variety of real-world situations. For example, it can be used to encode and decode user data.

To encode data, make a request to `/utils/encode` with a string as the body of the request. The endpoint will return the same string encoded in Base64.

To decode data, make a request to `/utils/decode` with a Base64 encoded string as the body of the request. The endpoint will return the original decoded string.
