# Currency Converter

This repository contains a Python script that converts one currency into another currency using the exchange rates obtained from the [Open Exchange Rates API](https://openexchangerates.org/).

## Instructions

To run the script, follow these instructions:

1. Clone the repository or download the script.
2. Get an API key from the [Open Exchange Rates website](https://openexchangerates.org/signup).
3. Replace "`YOUR_APP_ID`" in the `currency_converter.py` script with your API key obtained from step 2.
4. Install requirements with `pip install -r requirements.txt`.
5. Run the script with `python currency_converter.py`.

The script will ask the user for their base currency, target currency and amount they want to convert. The result will be displayed in the console.

## Requirements

- Python 3.x
- requests (`pip install requests`)
