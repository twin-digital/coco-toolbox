# Codecademy Data Analysis Capstone Projects

This repository contains the capstone projects completed by me as a part of the Codecademy Data Analysis Path. Each project folder contains the project details, data files, and the Jupyter notebook where the analysis has been performed.

## Project List

1. [US Medical Insurance Costs](/us-medical-insurance-costs)
2. [YouTube Trending Videos Data Analysis](/youtube-trending-videos-analysis)
3. [OkCupid Data Analysis](/okcupid-data-analysis)

## Getting Started

To run the Jupyter notebooks in this repository, you must have Jupyter Notebook installed on your system. You can download and install Jupyter Notebook from [here](https://jupyter.org/install).

You can download the project files by running the following command in your terminal:

```shell
git clone https://github.com/USERNAME/codecademy-data-analysis-capstone.git
```

Replace `USERNAME` with your GitHub username.
